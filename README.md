## 🚗 Safely App — Crowdsourced Road Safety
> **Status:** Submitted for **GEMASTIK 2026** Selection (UX Design Division)

### Overview
**Safely** is a navigation application prototype that doubles as a passive pothole detector. By leveraging a smartphone's built-in **Accelerometer** and **Gyroscope** sensors, the app maps out damaged roads in real-time. The primary goal is to generate crowdsourced data that urges local governments to accelerate infrastructure repairs.

### Key Highlights
* **Live Navigation:** Standard A-to-B routing interface.
* **Passive Detection:** Automatically flags uneven road anomalies via motion sensors.
* **Civic Impact:** Aggregates user trip data into a transparent road-quality map.
