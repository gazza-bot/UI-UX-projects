## Safely App - Prototype
`Submitted for Selection of GEMASTIK 2026 for UX Design Division`
This is a prototype of an app that can be used to navigate like Google Maps, but can detect potholes by using Accelerometer Sensor and Gyroscope Sensor. The main purpose of this App is to encourage the government to repair the holes at the street by crowdsourcing.
